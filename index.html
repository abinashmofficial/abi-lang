Local Installation Guide

Run the AbiLang compiler CLI and local web playground on your machine.

Option A: One-Click Automated Installer (Recommended) 
# Fetch and execute bash auto-installer script
curl -fsSL https://raw.githubusercontent.com/abinashmofficial/abi-lang/main/install.sh | bash
Or run locally: chmod +x install.sh && ./install.sh
Option B: Download Installer and Run Locally 
# 1. Download installer script using wget and execute
wget https://raw.githubusercontent.com/abinashmofficial/abi-lang/main/install.sh && chmod +x install.sh && ./install.sh

# 2. Compile TypeScript binary & register global terminal alias
npm run web
3. Run Globally from Anywhere 
# Launch interactive REPL console abi

# Run an external script file
abi script.abi
4. Local Playground Server 
npm run web
Option A: PowerShell Automated Installer (Recommended) 
# Run PowerShell installer remotely
Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/abinashmofficial/abi-lang/main/install.ps1'))
Or run local clone script: .\install.ps1
Option B: Manual Compilation & Global CLI Installation 
# 1. Clone repo, enter folder & install dependencies
git clone https://github.com/abinashmofficial/abi-lang.git
cd abi-lang
npm install

# 2. Compile TypeScript binary & register global terminal alias
npm run build
npm link
3. Run Globally from Anywhere 
# Launch interactive REPL console
abi

# Run an external script file
abi script.abi
4. Local Playground Server 
npm run web
Option A: Native Wrapper App via Capacitor (Visual Playground) 
# 1. Add Capacitor core and CLI tooling
npm install @capacitor/core @capacitor/cli

# 2. Initialize configuration schema
npx cap init AbiLang com.abinash.abilang --web-dir=web

# 3. Add Android and iOS platform wrappers
npm install @capacitor/android @capacitor/ios
npx cap add android
npx cap add ios

# 4. Sync assets & launch project inside Android Studio / Xcode
npx cap sync
npx cap open android
npx cap open ios
Builds and runs the interactive playground as a native Android or iOS application.
Option B: Embedded Core Interpreter inside React Native 
// 1. Copy web/dist/abilang.min.js to your React Native assets/
// 2. Load the bundle in your JavaScript files:
import './assets/abilang.min.js';

// 3. Define native device inputs/outputs:
const mobileIO = {
    print: (msg) => Alert.alert("AbiLang Output", msg),
    input: (prompt) => promptUserTextField(prompt)
};

// 4. Run scripts asynchronously:
const interpreter = new global.AbiLang.Interpreter(mobileIO);
await interpreter.interpret(statements);
Imports the raw execution core straight into mobile runtime engines (Hermes / JavaScriptCore).
